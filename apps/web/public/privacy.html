プライバシーポリシー

すっきりワリカン（割り勘精算アプリ）

すっきりワリカン（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが取得・利用する情報の取扱いについて定めるものです。

1. サービス提供者

サービス名 すっきりワリカン
提供者 すっきりワリカン 運営者
お問い合わせ アプリ内のお問い合わせ機能、またはLINE公式アカウントよりご連絡ください

2. 取得する情報

本アプリは、LINE プラットフォーム（LIFF SDK）を通じて、以下の情報を取得します。

情報の種類 取得方法 利用目的
LINE ユーザーID LIFF SDK（LINE ログイン） ユーザーの識別・認証
表示名 LINE プロフィール API アプリ内でのメンバー表示
プロフィール画像 URL LINE プロフィール API アプリ内でのアバター表示
取得しない情報: 本アプリは、LINE のトーク内容、友だちリスト、電話番号、メールアドレス、位置情報などは一切取得しません。

3. 情報の利用目的

取得した情報は、以下の目的にのみ利用します。

  1. ユーザー認証: LINE アカウントによるログイン・本人確認
  2. サービス提供: グループ内の支払い記録、精算計算、残高表示、振込みタスク管理
  3. 通知: 支払い報告・承認依頼・差戻し通知・精算完了通知の LINE プッシュメッセージ送信
  4. 表示: アプリ内でのメンバー名・アバターの表示

4. 情報の保存

取得した情報は、以下のとおり保存します。

  • 保存先: Supabase（PostgreSQL）上のデータベース
  • 保存期間: アカウント削除が行われるまで
  • 暗号化: 通信は TLS（HTTPS）で暗号化されます

5. 第三者への提供

本アプリは、取得した個人情報を第三者に提供しません。ただし、以下の場合を除きます。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき

利用する外部サービス

サービス 提供元 用途
LINE Messaging API LINEヤフー株式会社 プッシュ通知の送信
Supabase Supabase Inc. データベースホスティング
Vercel Vercel Inc. アプリケーションホスティング

6. グループ内での情報共有

本アプリの性質上、同一グループに参加しているメンバー間で以下の情報が共有されます。

  • 表示名・プロフィール画像
  • 支払い報告の内容（金額、説明、立替者、内訳）
  • 精算結果（振込み先・金額）
  • 承認・差戻しの操作履歴

これらの情報は、グループ外のユーザーには公開されません。

7. アカウント削除（退会）

ユーザーはいつでもアカウントを削除できます。削除時には以下の処理が行われます。

  • 表示名が「退会済みユーザー」に変更されます
  • プロフィール画像が削除されます
  • すべてのグループメンバーシップが無効化されます
  • LINE プラットフォームの認可権限が取り消されます
精算履歴など、他のメンバーに関わるデータは、サービスの整合性を保つため匿名化された状態で保持される場合があります。

8. Cookie・ローカルストレージ

本アプリは、以下の目的でブラウザのローカルストレージおよびセッションストレージを使用します。

  • グループ ID: 最後に使用したグループの記憶（再訪時の利便性向上）
  • ユーザー ID: セッション中の認証情報の保持

トラッキング目的の Cookie やアクセス解析ツールは使用していません。

9. 未成年者の利用

本アプリは、未成年者の個人情報を意図的に収集しません。未成年者が本アプリを利用する場合は、保護者の同意のもとご利用ください。

10. プライバシーポリシーの変更

本ポリシーの内容は、法令の変更やサービス内容の変更に応じて、事前の通知なく改定されることがあります。改定後のポリシーは、本ページに掲載した時点から効力を生じます。

11. お問い合わせ

本ポリシーに関するお問い合わせは、アプリ内の機能またはLINE公式アカウントよりご連絡ください。
